# Ardunio-DHT11

This is project of an Arduino using temperature and humity sensor DHT11.

## Notes :books:
- Responsive project to ambient.
- Practice electronics skills.

## Components :point_up:
- Arduino Uno x1.
- DHT11 Sensor x1.
- Power supply from own Arduino.
- Protoboard x1.
- Jumpers xN.

## Libraries :bookmark_tabs:
- Adafruit_Sensor
- DHT
- DHT_U
